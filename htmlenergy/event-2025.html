What is this?

This is a HTML freewrite, an event where people get together to (learn and) write HTML documents!

HTML documents form the backbone of every website you've ever visited. Before website builders (Squarespace, Wix, etc), people made websites by coding their own HTML documents.

In the spirit of HTML Energy, everyone will be handcoding a HTML document during this freewrite event.

Who can come?

Literally anyone ‚Äî no coding skills required! Everyone can learn to code in HTML. Most importantly, we'll help each other out.

RSVP is not required, but is helpful for me to gauge attendance!

Where will it be?

Exact coordinates: https://maps.app.goo.gl/2DTrPZA5xcW3YJFp7

What should I bring?

Essentials:
  • Yourself
  • Something to write HTML on (e.g., fully charged laptop [recommended], mobile phone, pen & paper)
  • Water bottle with water
  • Umbrella, hat and/or poncho
  • Good vibes 8-)
Optionals:
  • A friend, or two
  • A fan, if you get warm
  • A picnic mat, if you have one
  • Snacks and/or beverages to share
  • Ideas for HTML page(s)

What should I expect?

Meeting new people, asking for help and helping others, creating something for fun! We'll be writing and coding in HTML and CSS primarily. You may use JS if you want to!

Here's a tentative schedule:

4.00pm Set-up
4.15pm Basic HTML tutorial and/or ideation
4.40pm Start coding
6.10pm Sharing session (for those comfortable)
6.30pm Event end, but feel free to continue coding!

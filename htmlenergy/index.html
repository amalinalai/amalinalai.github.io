HTML freewrites are event where people get together to (learn and) write HTML documents – check out HTML energy for the philosophy behind it & events all over the world!

What is HTML?

HTML documents form the backbone of every website you've ever visited. Before website builders (Squarespace, Wix, etc), people made websites by coding their own HTML documents.

Can I join? I've never coded before.

Yes!! Everyone can learn to code in HTML. It's a very forgiving and rewarding coding language to play with. And we'll help each other out! There's a starter kit if it's your first time!

More resources

Tons of people before me have created amazing resources (or compilation of resources) for starting out with coding in HTML (and CSS). Here are just a selection of them and is by no means comprehensive!

More compilations

  • List of resources (HTML in Hyde)
    A comprehensive list of guides, resources and inspiration for writing HTML and CSS by Max, who hosts the HTML in Hyde (Sydney) freewrites. The list is detailed and organized into multiple sections.
  • "Make Yourself a Website" by Lizbeth Poirier
    Lizbeth talks a little about why you should make your own website. Included is a list of some inspiration and resources to check out.

Guides & Templates

  • W3schools HTML tutorials and CSS tutorials
    W3schools provides long and comprehensive tutorials. My favourite part about them is that they have a 'Try it Yourself' feature that allows you to edit their code within the browser and see how the website changes for yourself. It's a fun resource that allows for experimentation and learning by trying.
  • Cheatsheet
    A cheatsheet shared by the Melissa, Rizq, and Phoenix for their 2024 London freewrite.
  • Flexbox Froggy
    A interactive tutorial for CSS flexboxes. Requires basic CSS and HTML knowledge.
  • 1-line layouts
    1-line layouts is a resource that is geared more towards people who have coded in HTML before. The website provides a quick styling for divs that you can simply copy and paste.
  • stealing rocks
    As they put it: A collection of stolen layouts and components from popular web-builders recreated in pure HTML/CSS/JS for you to steal for your own site.

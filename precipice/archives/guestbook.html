guestbook

aka leave your mark!

A guestbook was something I remember being too afraid to use whenever I saw it on websites when I was younger. I don't know, somehow it was more daunting than Chatterbox, but maybe it's an exposure thing. Anyway, I'm less afraid now, and I wanted to install a little guestbook of my own for people who happen to chance across my site to leave a note!

Unfortunately, Github pages (my current host), only allows for static pages, so traditional php guestbooks weren't going to work. I had to use some other solution, and thankfully people smarter than me had already figured it out! Despite the fact that I wanted this website to be mostly HTML & CSS based, unfortunately Javascript still allows for some pretty nifty, interaction-based things, so I had to concede for this page.

version 1.0 › august 2024 — ???

Creating the guestbook took longer than I expected, to be honest. I originally was going to use Vera/GroundedWren's guestbook framework, but it required manually transferring information from a Discord channel to a Google sheets as a means to verify information, which I thought would be too cumbersome. Instead, I knew that Google forms would be a much simpler way to collect information directly, and I also knew that it was possible to somehow filter out entries from one sheet to another automatically using Google Sheet formulas.

I want to say here: big thank you to my dad who helped me with filters on Google Sheets. He's an Excel whiz, and he was pretty excited to be helping me solve my little problem, which I'm happy about! (He even asked me after if I had anything else I needed help with on Excel, how cute.)

Originally, I meant to modify GroundedWren's guestbook code to fit in the Google sheets element to the form, but because of my lacklustre Javascript knowledge, I didn't really get far. That's when, after some searching, I found Kiki/Virtual Observer's code for a guestbook that uses Google forms as a means of collecting input. However, it didn't have a moderation feature (if I had read a little more... I would've seen someone else provide a guide for moderation...), but it was pretty quick to change the original code to look at a specific sheet of a Google Sheets document, thanks to inspecting GroundedWren's Google Sheets reader!

After a whole load of tinkering, and reading Kiki's code to understand Javascript syntax and what was going on, I finally added some extra code to have a little alert pop up after someone submits a message to provide some feedback and affirmation that their message has been received, since new comments don't automatically show up immediately anymore due to the moderation feature.

Glad to say I understand Javascript a tiny bit better now!

Sorry I'm yapping so much, but we're finally talking about the design! I'm actually super proud of how it looks, I used the border-image property I learnt from making my "about me" V1.0 page to make the guestbook comments look like little airmail envelopes! I made all the pixel art for the papers, airmail envelope and stamp using the 70s Zine Palette by polyducks. At some point I'd love to make more complex designs, e.g. the 'stamped' ink when a letter gets sent through post.
